# Investigating Netflix Movies

## Overview
This project analyzes a dataset of Netflix movies to explore trends in movie duration, release years, genres, and other characteristics.

## Skills Demonstrated
- Python
- Pandas
- Data Cleaning
- Data Visualization

## Files
- `notebook.ipynb` — Jupyter Notebook containing the analysis
- `netflix_data.csv` — Dataset used in the project

## Project Source
Completed as part of a DataCamp learning track.

## Key Questions
- What was the most frequent movie duration in the 1990s? 
- The number of short action movies released in the 1990s.

## Technologies Used
- Python
- Pandas
- Matplotlib
